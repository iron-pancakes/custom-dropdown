# Custom Dropdown with only HTML and CSS

This is a custom dropdown with only html and css.   
it uses the lastest features for chrome to easily override the native os dropdown styling.


It is for https://roadmap.sh/projects/custom-dropdown

You can learn more about customizable select on the [Mozilla website](https://developer.mozilla.org/en-US/docs/Learn_web_development/Extensions/Forms/Customizable_select)
